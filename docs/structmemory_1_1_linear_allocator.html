Detailed Description

Linear (Arena) allocator implementation.

A linear allocator allocates memory in a continuous fashion without support for individual deallocation. Memory is typically freed all at once. Also known as Arena or Region-based allocator.

Note
This allocator has O(1) allocation complexity.
