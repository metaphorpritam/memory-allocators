Detailed Description

template<typename T>
struct memory::DynArray< T >

Dynamic array implementation with custom allocator support.

A dynamic array that can grow as needed and supports custom allocators. Provides standard container operations with error handling via std::expected.

Template Parameters
TThe type of elements stored in the array
